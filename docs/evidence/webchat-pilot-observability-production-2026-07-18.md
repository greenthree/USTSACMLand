# WebChat 试运行观测生产验收（2026-07-18）

## 验收范围

本次只增加管理员试运行观测能力，不扩大成员权限，也不保存或读取对话正文。管理员可在 `/admin/webchat` 查看所有已显式配置 AI 权限的账号及以下聚合数据：

- 姓名、年级、专业、角色和账号状态。
- 权限开关、每日请求/Token 上限、配置版本和更新时间。
- 北京时间今日请求数、已结算/预留 Token、剩余额度和活动请求数。
- 最近一次请求时间，以及前往成员详情调整权限的入口。

页面不展示邮箱、QQ、请求编号、请求指纹、Prompt、回复或系统提示词。

RPC 只列出 `private.webchat_member_access` 中已显式配置的账号，包括权限已关闭或账号已停用的记录；未配置账号不会被静默加入试运行名单。

今日用量口径与成员端 `read_own_webchat_usage()` 一致：

- 过期且尚未开始上游请求的 `claimed` 租约退回请求次数和预留 Token。
- 已开始上游请求但租约过期的 `started` 请求保留请求次数，并按预留 Token 计入已结算用量。
- 活动请求只统计租约尚未过期的 `claimed` / `started` 请求。
- 最近请求时间只取历史 `claimed_at` 最大值，不读取请求内容。
