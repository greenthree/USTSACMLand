# USTSACMLand 界面设计规范

当前环境无法调用内置图像生成工具，因此本文件作为首版可执行视觉规范。实现完成后以浏览器截图迭代校正。

## 设计方向

- 产品类型：面向校队成员和管理员的安静、数据密集型工作界面，不采用营销落地页结构。
- 背景：接近白色的冷灰 `#f5f7f6`，主要工作区为白色；深色导航使用墨绿黑 `#0b1912`。
- 主色：校队绿 `#16794b`；强调色使用竞赛红 `#d74d3f` 和排名金 `#c58a1b`，避免单一色相。
- 字体：中文使用系统无衬线，数字使用等宽数字特性；字距固定为 `0`。
- 密度：桌面端中高密度，移动端保持可扫描，表格在窄屏转换为有明确列标签的列表行。
- 容器：页面采用全宽工作区、侧栏、工具条和表格；只对重复的成员条目和模态框使用卡片，不嵌套卡片。
- 圆角：控件 6px，面板最多 8px。

## 首版页面

1. 公开榜单：顶部品牌与导航、数据更新时间、Rating/刷题双视图、平台切换、搜索、专业与年级筛选、成员排名表。
2. 成员详情：身份摘要、平台账号、最新值/最高值、数据新鲜度和趋势占位区。
3. 登录注册：简洁双栏表单，移动端单栏；不展示额外营销文案。
4. 个人资料：基本资料、平台绑定、审核状态和同步状态。
5. 管理后台：左侧导航，概览指标、待审核成员、同步健康度、日志表格。

## 控件与图标

- 所有工具按钮使用 Lucide 图标：搜索、筛选、刷新、用户、设置、退出、审核、重试。
- 模式使用分段控件，平台使用标签页，布尔状态使用开关或复选框。
- 不使用文字箭头或手绘 SVG；未知图标必须有 `title` 或可访问名称。

## 允许的首屏文案

- `USTSACMLand`
- `苏州科技大学 ACM 集训队`
- `榜单`、`成员`、`我的资料`、`管理后台`、`登录`
- `Rating 榜`、`刷题榜`
- `搜索成员`、`全部专业`、`全部年级`、`数据更新于`

## 响应式约束

- 桌面内容最大宽度 1440px，左右最小间距 24px。
- 低于 900px 时隐藏次要导航文字并收紧表格列。
- 低于 680px 时工具条换行、表头隐藏、每行显示字段标签，任何按钮和长用户名不得溢出。
- 固定格式控件使用稳定高度，切换状态不引发布局跳动。
