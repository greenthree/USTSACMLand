# USTS ACM Land 界面设计规范

本文记录当前站点的可执行视觉规范；界面变更需通过桌面端与移动端浏览器截图迭代校正。

## 设计方向

- 产品类型：苏州科技大学 ACM 集训队官网；首页承担竞赛介绍、训练资源与招新说明，榜单和后台保持安静、数据密集的工作界面。
- 背景：接近白色的冷灰 `#f5f7f6`，主要工作区为白色；深色导航使用墨绿黑 `#0b1912`。
- 主色：校队绿 `#16794b`；强调色使用竞赛红 `#d74d3f` 和排名金 `#c58a1b`，避免单一色相。
- 字体：中文使用系统无衬线，数字使用等宽数字特性；字距固定为 `0`。
- 密度：桌面端中高密度，移动端保持可扫描，表格在窄屏转换为有明确列标签的列表行。
- 容器：页面采用全宽工作区、侧栏、工具条和表格；只对重复的成员条目和模态框使用卡片，不嵌套卡片。
- 圆角：控件 6px，面板最多 8px。

## 首版页面

1. 官网首页：介绍 ACM、校内外主要赛事、线上公开赛平台、学习资源、公开训练记录与年度入队比赛。
2. 公开榜单：顶部品牌与导航、数据更新时间、Rating/刷题双视图、平台切换、搜索、专业与年级筛选、成员排名表。
3. 成员详情：身份摘要、平台账号、最新值/最高值、数据新鲜度和真实历史趋势。
4. 登录注册：简洁双栏表单，移动端单栏；注册时直接填写姓名并在成功后进入资料页。
5. 个人资料：基本资料、平台绑定、验证状态、同步状态、修改密码与自助注销。
6. 管理后台：左侧导航，成员管理、平台绑定、手工数据、同步中心、数据源健康、公告与审计日志。

## 控件与图标

- 所有工具按钮使用 Lucide 图标：搜索、筛选、刷新、用户、设置、退出、验证、重试。
- 模式使用分段控件，平台使用标签页，布尔状态使用开关或复选框。
- 不使用文字箭头或手绘 SVG；未知图标必须有 `title` 或可访问名称。

## 允许的首屏文案

- `USTS ACM Land`
- `苏州科技大学 ACM 集训队`
- `首页`、`榜单`、`我的资料`、`管理后台`、`登录`
- `Rating 榜`、`刷题榜`
- `搜索成员`、`全部专业`、`全部年级`、`数据更新于`

## 响应式约束

- 桌面内容最大宽度 1440px，左右最小间距 24px。
- 低于 900px 时隐藏次要导航文字并收紧表格列。
- 低于 680px 时工具条换行、表头隐藏、每行显示字段标签，任何按钮和长用户名不得溢出。
- 固定格式控件使用稳定高度，切换状态不引发布局跳动。
